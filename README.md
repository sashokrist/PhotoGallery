Laravel 10, PHP 8.1, Bootstrap 4.1, jquery/3.6.4

How to install and run:

1. git clone git@github.com:sashokrist/PhotoGallery.git

2. composer install --ignore-platform-req=ext-gd

3. cp .env.example .env

4. php artisan key:generate

5. Set database credentials in .env

6. php artisan migrate --seed

7. npm run dev

8. php artisan serve
